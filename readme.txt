#Notas
```
Para Ejecutar el ejemplo siguiente comando
node app -b 7 -l  -h 20
-b es el numero base por el cual se va a multiplicar
-l si quiere que salga la tabla
-h para decir hasta que numero quiere que se multiplique
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Es la base de la tabla de multiplicar       [number] [required]
  -l, --listar   Es la base mostrar la tabla de multiplicar
                                           [boolean] [required] [default: false]
  -h, --hasta    hasta que numero quiere que se multiplique  [number] [required]
```
